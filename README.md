# 👶 ShishuSneh

A baby health tracking Android app built with Kotlin and Material Design.

## 📱 Features

- **Baby Profile** — Store baby's name and age with date of birth calculator
- **Growth Tracker** — Track weight and height with history of last 3 entries
- **Vaccination Tracker** — Mark and track BCG, Hepatitis B, and Polio vaccines
- **Dashboard** — Overview of all data with vaccine progress bar
- **Smart Greeting** — Dynamic greeting based on time of day

## 🛠️ Tech Stack

- **Language** — Kotlin
- **UI** — XML Layouts with Material Design 3
- **Storage** — SharedPreferences
- **Architecture** — Fragment-based navigation
- **Min SDK** — 24 (Android 7.0)
- **Target SDK** — 34

## 🚀 Getting Started

1. Clone the repo — https://github.com/reddy-varshini9/ShishuSneh.git
2. Open in Android Studio
3. Build and run on any Android device (API 24+)
